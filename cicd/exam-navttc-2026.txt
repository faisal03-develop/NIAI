Task 1: Core Python Programming (20 minutes)

Write a Python program that:

Takes a list of integers as input
Removes duplicates
Sorts the list in ascending order
Prints:
The final list
The maximum and minimum values
The average of the numbers

👉 Bonus: Handle invalid input (e.g., strings in the list)

Task 2: Object-Oriented Programming (25 minutes)

Create a class called Student with the following:

Attributes:
name, roll_number, marks (list of scores)
Methods:
calculate_average()
get_grade() (A, B, C, Fail based on average)
display_info()

Then:

Create at least 2 student objects
Display their details and grades

👉 Concepts covered: encapsulation, methods, class design

Task 3: Advanced OOP Concepts (20 minutes)

Demonstrate:

Inheritance
Create a base class Vehicle
Create a derived class Car
Polymorphism
Define a method start_engine() differently in both classes
Method Overriding

👉 Write a short explanation (2–3 lines) of how polymorphism is used.

Task 4: NumPy Operations (25 minutes)

Using the NumPy library:

Create a 3x3 array with random integers
Perform:
Transpose of the array
Find mean, median, and standard deviation
Reshape into a 1D array
Replace all values greater than 50 with 100

👉 Bonus: Explain why NumPy is faster than Python lists

Task 5: Pandas Data Analysis (30 minutes)

Using Pandas:

Create a DataFrame with:
Columns: Name, Age, Marks
Perform the following:
Display first 3 rows
Filter students with marks > 70
Add a new column Status (Pass/Fail)
Calculate average marks
Sort the DataFrame by marks descending

👉 Bonus: Export the DataFrame to a CSV file
