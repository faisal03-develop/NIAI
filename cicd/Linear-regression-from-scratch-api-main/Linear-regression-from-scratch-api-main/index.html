Under the hood
How it works — A to Z
From raw data to a live prediction in your browser — every step explained
1

Generate data

1000 synthetic house records created with NumPy. Features: size, bedrooms, age, distance from city.

2

Feature scaling

Z-score standardisation — each feature scaled to mean=0, std=1 so gradient descent converges fast.

3

Train model

Linear Regression from scratch using NumPy. 1000 iterations of Gradient Descent. Loss falls from 1.0 → 0.01.

4

Save weights

Trained weights and scaler saved to models/linear_regression.pkl and scaler.pkl using pickle.

5

Flask API

app.py loads the .pkl file once at startup. Every POST /predict call loads your features, scales them, and runs ŷ = X·W + b.

6

This website

JavaScript fetch() sends your slider values to the API and displays the prediction — no page reload needed.
